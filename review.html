Sun Microsystems



Sun Microsystems was the company behind the development of the well-known Java programming language, 
as well as other products and services. As of the summer of 2010, Oracle has acquired Sun Microsystem, Inc.1

Hardware


Since the late 1980s Sun Microsystems have used its own design of an RISC processor architecture, when they created 
the SPARC. Since 2007 Sun used the updated form of their design in the UltraSPARC T2 Plus, which has an 8-core chipset, 
64 bit computing rate, 2 memory controllers,up to 16 or 32 FB-DIMMs.2  What benefits that came with the RISC processor architecture was that RISC 
was able to scale up in performance quite quickly and cheaply. Sun also paired up their processors with the Intel 80386-based machine, 
this was designed to be a hybrid system, running Sun OS but at the same time supporting DOS applications.3 

Open Source


One fully open source implementation of the SPARC architecture was
LEON, which is a 32-bit, SPARC Version 8 implementation, designed especially for space use. 
The Source code is written in VHDL, and licensed under the GPL.4



Software


Sun’s UNIX operating system, Solaris, was “developed as proprietary software”, but Sun released 
“most of the software’s codebase under the CDDL license.”5
On Sun’s SPARC system, Solaris was known for its scalability and had a “reputation for being well-suited to symmetric multiprocessing, supporting a large number of CPUs.”6
In order to allow programs to function regardless of the device they were used on, Sun developed and ran Java, an object-oriented programming language; one of the world’s most 
popular programming languages. 


Open Source/ Special Software


Openoffice: Open sources office suit which was written in C++ and Java and was for the Linux, OS X, Microsoft Windows, and Solaris OS. 
Openoffice was created on the IA-32, x86-64, PowerPC, and SPARC platforms. The size of the suit is 143.4 MB.



Servers


Web Servers


The Sun Fire X4500 coincided with the UltraSPARC iii processors. The Sun Fire X4500 server maintains two dual-core AMD Opteron processors and up to 64 GB RAM. 
With 48 500/1000/2000 GB SATA drives, it provides 24/48 TB of storage in 4 rack units.7


 Database Servers


For their database purposes Sun “acquired MySQL AB, the developer of the MySQL database in 2008 for 1 billion US dollars.”8 


Deployment Strategies & Security


Although at this point Sun has been acquired by Oracle, their security strategies have been adopted by Oracle. The most current implementation of security strategy was Oracle Application Server 
on Oracle Identity Management. 
“Installing and deploying Oracle Application Server components involves creating identities for the applications being deployed and granting them run-time privileges to necessary resources, such as Oracle Application Server infrastructure schema, 
and access to other application components. Without proper delegation, deployment of any application would require the directory administrator to be involved. On the other hand, with excessive privilege delegation, 
an administrator with privileges to deploy one application will also have unwarranted privileges over other applications. 
With proper delegation, specific administrators can be de granted privileges to specific applications.”9
